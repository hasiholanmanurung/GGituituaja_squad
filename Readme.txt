Yang dihasilkan pada proyek ini adalah:

1. Bar Chat untuk mengetahui Tag yang sering dibawakan dalam Event Organizer
2. Bar Chat untuk menentukan jenis Tag yang dapat dibawakan oleh pembicara (Speaker)
3. Bar Chat untuk menentukan banyaknya Tag yang dapat ditentukan oleh Event Organizer

Untuk menghasilkan setiap Bar Chat dalam sebuah proyek ini, jumlah viewer sudah ditentukan yaitu sebanyak 100.000 viewer.
Untuk pembaca proyek ini, pembaca harus memiliki tools berupa Tableau.
